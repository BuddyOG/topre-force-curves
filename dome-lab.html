How the lab works

Measured, not quoted

Every force figure comes off a calibrated instrument at 0.005 mm steps, averaged across runs. Dome-to-dome variation is measured too: multiple physical domes of one type are tested and grouped, never blended silently. If it wasn't measured here, it's labeled.

One source of truth

The parts and dome catalog is a single confirmed dataset; the tools read it verbatim. Names come from the raw list — never vendor terminology — and vendor claims are never treated as measurements. Who's behind it.

Open by default

Raw CSVs, the dataset, and all three tools live in a public repo. New bench sessions flow through one importer — validated, outlier-checked, exported — and every tool picks them up automatically, with the data revision stamped in the library header.

Credit where due

The force-curve instrument is a build of bluepylons' open-source Open-Switch-Curve-Meter. Sources, patents, and designs are cited, always.
